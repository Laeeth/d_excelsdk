Xlw User Contributed Files - dlib
===================================

http://dlib.net/

Demonstrates using an alternative matrix library.


Build instructions gcc & Code::Blocks
=====================================

Set the enviromment variable DLIB_ROOT to the location
of the unpacked dlib library

Build instructions Visual Studio
=====================================

Add the root directory of your dlib install to the 
include directories for Win32 (and x64 if present)
